# AnimeDesktop

### [下载链接](https://github.com/zhmushan/AnimeDesktop/releases)

这是一个摆放在桌面上的玩具, `electron` 打包的 `UWP`, 抄自 https://github.com/amorist/platelet, 按住 `Ctrl` 可以到处拖动.
